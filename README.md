# Example store

В этом репозитории находится приложение-тренажер для написания автотестов.

Проект имитирует реализацию интернет-магазина, но код написан в упрощенном, учебном стиле. Приложение написано на typescript+react со сборкой на vite, есть серверная часть. Использованы популярные библиотеки: [react-router](https://www.npmjs.com/package/react-router), [react-query](https://www.npmjs.com/package/@tanstack/react-query), [redux-toolkit](https://www.npmjs.com/package/@reduxjs/toolkit).

Для проекта описаны [функциональные-требования](#функциональные-требования). Вы можете попробовать написать для них автотесты.

## Как запустить

```sh
# установите зависимости
npm ci

# запустите сервер
npm run dev
```

После этого можете открыть приложение в браузере по адресу http://localhost:5173

## Функциональные требования

**Общие требования:**

- название магазина в шапке должно быть ссылкой на главную страницу
- в шапке отображаются ссылки на страницы магазина: каталог, контакты
- в шапке должно отображаться текущее количество товаров в корзине
- при клике на элемент корзины в шапке происходит переход на оформление заказа
- содержимое корзины должно сохраняться между перезагрузками страницы

**Каталог:**

- в каталоге должны отображаться товары, список которых приходит с сервера
- для каждого товара в каталоге отображается название и цена
- название является ссылкой на страницу c подробной информацией о товаре 
- для каждого товара отображается кнопка "Купить", при нажатии товар добавляется в корзину
- если товар уже добавлен в корзину, повторное нажатие кнопки "добавить в корзину" должно увеличивать его количество

**Страница товара:**

- отображаются название товара, его описание, цена, цвет, материал
- отображается кнопка "добавить в корзину", при нажатии товар добавляется в корзину
- если товар уже добавлен в корзину, повторное нажатие кнопки "добавить в корзину" должно увеличивать его количество

**Оформление заказа:**

- ✅ должна отображаться таблица с добавленными в нее товарами
- ✅ для каждого товара в таблице должны отображаться название, цена, количество, стоимость
- в последней строке таблицы должна отображаться общая сумма заказа
- должна быть кнопка "очистить корзину", по нажатию на которую все товары удаляются
- ✅ если корзина пуста, должна отображаться ссылка на каталог товаров
- ✅ для оформления заказа нужно ввести ФИО, телефон, адрес, все поля обазательны для заполнения
- ✅ при нажатии кнопки "Заказать" содержимое корзины и значения полей формы заказа отправляются на сервер и оформляется новый заказ
- после оформления заказа на экране отображается номер созданного заказа
